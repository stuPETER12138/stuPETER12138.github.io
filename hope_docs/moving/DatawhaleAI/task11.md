task 1[^1]: 从零入门 AI 逻辑推理

### 运行 baseline

为了保证顺利体验大模型，同时满足比赛要求，baseline 采用阿里开源大模型 Qwen 系列，通过 api 的形式调用，使用的模型是`qwen1.5-1.8b-chat`模型。

跑代码大概需要 30 分钟，最终成绩 0.3494。

### 那么，来聊聊赛题

本次学习活动的赛题“复杂推理能力评估”是一个**大语言模型处理推理任务的问题**。

#### 1. 关于数据集

本次活动官方的初赛数据集属于**逻辑推理数据集**，其中训练集和测试集各 500 条数据。每条数据包括若干子问题，每个子问题为单项选择题，选项不定（最多5个）。我们的目标是**为每个子问题选择一个正确答案**。推理答案基于闭世界假设（closed-world assumption），即未观测事实或者无法推断的事实为假。

对于 baseline 的测试集`round1_train_data.jsonl`，包含内容如：

```json
{"problem": "有几位学生参与了一门人工智能课程的考核，考核包括两次家庭作业和两次考试。以下是每位学生通过的考核情况：\n\n1. Mia 通过了第一次家庭作业、第二次家庭作业、第一次考试和第二次考试。\n2. Max 通过了第一次家庭作业、第二次家庭作业、第一次考试，还通过了一次补充考试。\n3. Paul 通过了第一次家庭作业、第二次考试和补充考试。\n\n在这门课程中，通过所有家庭作业和至少两次考试（包括补充考试）的学生将被视为通过了整个课程。根据以上信息，回答以下选择题：", 
"questions": [
    {"question": "选择题 1：\n哪位学生通过了人工智能课程？", "options": ["Mia,Max", "Max,Paul", "Paul"]}, 
    {"question": "选择题 2：\nMax 是否通过了人工智能译程？", "options": ["是", "否"]}, 
    {"question": "选择题 3：\nMia 是否通过了人工智能课程？", "options": ["是", "否"]}], 
    "id": "round1_test_data_136"}
```

初赛数据集会额外包含`answer`字段。除此之外，参赛选手**可以使用开源数据**，若使用了开源数据，需要在代码审核阶段提供数据来源以及相关说明。

#### 2. 关于比赛任务

本次比赛提供基于自然语言的逻辑推理问题，涉及多样的场景，包括`关系预测`、`数值计算`、`谜题`等，希望选手通过分析推理数据，利用机器学习、深度学习算法或者大语言模型，建立预测模型，完成推理任务。

比赛期间，参赛队伍通过天池平台下载数据[^2]，本地调试算法，在线提交结果。比赛的评估指标为**所有子问题的回答准确率**，每个子问题权重相同。

预测结果文件格式如下：

```json
{'id': 'round1_test_data_000',
 'questions': [{'answer': 'A'}, {'answer': 'D'}, ...], # 顺序与子问题对应
}
```
